这个项目是一个基于C语言的协议分析器，用于分析网络数据包。只用于计算机网络教学，不用于任何商业用途。AI辅助实现。

## 开发环境

- 操作系统: Linux
- 编译器: GCC
- 库: libpcap
- 语言: C
- 工具: CMake

## 快速开始

1. 安装依赖库: libpcap

2. 编译项目: `mkdir build && cd build && cmake .. && make`
